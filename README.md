A project template for building Angular.js apps with [Lingon](https://github.com/jpettersson/lingon)

## Get started

**Install dependencies:**

```
npm install
bower install
```

**Use:**

```
Build once and quit:
./lingon.js build

Start the server: 
./lingon.js

Start the server on a custom port:
./lingon.js server -p 1111
```
